Features

Designed for focus and flexibility

Everything you need to manage tasks without the clutter: cross-platform support, offline-first design, and full control over how you organize your work

Runs seamlessly on Linux, Windows, and macOS with a system tray for quick access.

Customizable Task Management

Define task types, add tags, set priorities, and configure your own task table layout.
